# Chat Room

A real-time chat application built with Flask and WebSocket.

## 🎯 Features

- Real-time messaging
- Multiple chat rooms
- User authentication
- Message history
- Online status indicators
- Typing indicators

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/none-ai/chat-room.git
cd chat-room

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py
```

Visit http://localhost:5000

## 📋 Requirements

- Python 3.8+
- Flask
- Flask-SocketIO
- See `requirements.txt` for full list

## 🛠️ Tech Stack

- Python 3
- Flask
- Flask-SocketIO
- Socket.IO
- HTML/CSS/JavaScript
